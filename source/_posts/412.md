> sort 命令在 Linux 里非常有用，它将文本数据进行排序，并将排序结果标准输出。sort 命令既可以从特定的文件，也可以从 stdin 中获取数据。

<!-- more -->

### 字典排序

```
$ cat /etc/passwd | sort
```

### 反转排序

```
$ cat /etc/passwd | sort -r
```

### 特定排序

```
$ cat /etc/passwd | sort -t ':' -k 3
```

### 数值排序

```
$ cat /etc/passwd | sort -t ':' -k 3 -n
```
