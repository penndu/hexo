## 特性

1. 统计用户所发说说条数；

2. 显示用户头像；

3. 显示用户昵称；

4. 显示平台的用户名；

5. 支持大图显示；

6. 可一键至说说广场进行评论。

## 使用

请根据需求修改对应的内容：

| 参数 | 说明 |
| - | - |
| host | 域名 |
| limit | 每页显示条数 |
| creatorId | 用户的 ID |
| domId | 显示位置 |
| username | 广场的用户名 |
| name | 昵称 |
