## 命令介绍

```
gpasswd GROUPNAME
```

注意：之前篇章介绍过通过 `gpasswd` 管理群组成员，本篇文章主要通过 `gpasswd` 设置群组的密码，命令如上。

## 群组切换

```
newgrp GROUPNAME
```

注意：通过上面命令即可切换当前用户所在群组，不过只是暂时切换。需要注意的是，普通用户需要输入群组密码才能切换。

## 演示命令

```
mkdir cjk
# 创建目录
cd cjk/
# 进入目录
touch dusays
# 创建文件
ls -l
# 查看权限
groupadd dusays
# 添加群组
newgrp dusays
# 切换群组
gpasswd dusays
# 设置密码
useradd bdyjy
# 添加用户
```
