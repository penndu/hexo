## 操作步骤

```
yum -y install zsh
#下载并安装Zsh
yum -y install git
#下载并安装Git
sh -c "$(wget https://raw.sevencdn.com/penndu/oh-my-zsh/master/tools/install.sh -O -)"
#下载并安装Oh My Zsh
vim ~/.zshrc
#编写配置文件
ZSH_THEME="gnzh"
#将引号内的内容改为gnzh
```

注意：按照上面操作执行之后，重启系统打开终端即可看到上图效果！
