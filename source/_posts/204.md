> 经常更换管理密码可以一定程度防止服务器被暴力破解，不过很多管理员都懒得、或者是忘记改密码，有什么样的强制形式可以督促管理员修改密码？今天杜老师就说下 Linux 口令过期后账号最长有效天数策略！

<!-- more -->

### 漏洞名称

```
Linux 口令过期后账号最长有效天数策略
```

### 危险等级

```
低危
```

### 漏洞类型

```
安全基线
```

### 漏洞描述

```
设置口令过期后账号仍能保持有效的最大天数
```

### 修复方案

```
编辑/etc/default/useradd 文件配置 INACTIVE=365
```
