> 杜老师家里有一只橘白相间的猫，平时最爱跳上书桌"监工"。一开始杜老师挺困扰的，后来摸索出一套应对方案，今天就跟大家分享一下。

<!-- more -->

## 一、为什么不让猫上桌

杜老师一开始试过"坚决不让它上来"，但很快就放弃了：

* 猫的好奇心比任何训练都强
* 关上门它就在门口叫，更影响工作
* 把它赶下去 3 次，第四次它又跳上来

后来杜老师换了个思路：**让它上来，但让它上来之后没危险**。

## 二、3 个应对方案

### 方案 1：易碎品全部下桌

杜老师的桌面规则：

* 显示器、机械键盘可以留
* 水杯、收纳盒全部下桌
* 文件、纸质资料挪到书架

这样即使猫跳上来踩键盘，也踩不出大问题。

### 方案 2：给猫留一个"专属位置"

杜老师在桌面左侧放了一个软垫：

* 猫熟悉位置后，每次上来会先到软垫
* 工作间隙想摸它，伸手就到

注意：软垫要放稳，猫才会反复去。

### 方案 3：零食"勾引"下地

杜老师平时在桌下放一小盒猫粮：

* 工作时猫跳上来，喊它名字 + 拿零食
* 反复几次后，猫会主动下地"领赏"

这套方法的关键是**不要在它上来的时候大声吓它**。

## 三、杜老师自己的桌面长这样

```
+-----------------------+
|        显示器          |
+-----------------------+
|    手机竖立(副屏)       |
+-------+---------------+
| 猫    |    键盘+鼠标   |
| 软垫  |              |
+-------+-------+-------+
        | 透明  |
        | 水杯  |
        +-------+
            ↓
        桌下猫粮盒
```

猫在软垫上，键盘在工作区，猫粮在桌下。各得其所。

## 四、3 个方案的组合

杜老师最终的组合是这样的：

```
猫跳上来 → 软垫接住
猫想躺键盘 → 把它引导到软垫
猫饿了 → 零食引导下地
```

这套组合，杜老师用了半年多，目前人和猫都比较满意。

## 写在最后

养猫这事，杜老师觉得"硬挡不如引导"。让猫舒服，自己也舒服。
