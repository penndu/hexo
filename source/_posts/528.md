## 特性

1. 使用 Vue.js 全家桶开发；

2. 支持歌词显示；

3. 「各类音源」指默认启用的音源；

4. 每日自动签到「手机和电脑端同时签到」

5. 支持音乐云盘；

6. 自定义快捷键和全局快捷键。

## 部署

```
docker run -d --restart=always -p PORT:80 penndu/cloudmusic
```

注意：`-d` 在后台运行。`--restart-always` 关闭后自动重启，根据需要自行选择添加。`PORT` 改为自定义端口，可用于服务器反向代理。
