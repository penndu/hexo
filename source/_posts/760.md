## 特性

1. 显示用户头像；

2. 显示用户昵称与平台用户名；

3. 支持大图显示；

4. 可一键至说说广场进行评论。

## 使用

请根据需求修改对应的内容：

| 参数 | 说明 |
| - | - |
| host | 域名 |
| limit | 每页显示条数 |
| creatorId | 用户的 ID |
| domId | 显示位置 |
| username | 广场的用户名 |
| name | 昵称 |
